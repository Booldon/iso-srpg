# Element Mechanics — Confirmed Dual-Nature Design

**Status:** Confirmed  
**Elements:** Fire / Ice / Earth (3 total; no elemental cycle counter — see §4.1)  
**Design grammar:** All three elements share the pattern **stack accumulation → threshold effect**

---

## Fire — Destruction x Renewal

### Primary mechanic — Burn (destruction)
- Apply Burn stacks to enemies
- At stack threshold: detonate for burst STR damage
- Core feel: "charge up then explode"

### Secondary mechanic — Solar (renewal/creation)
- Fire as life-giving warmth, not only destruction
- Expressed as: temporary STR bonus (ties into §3.2 temp Strength system), post-burn regeneration
- Thematic basis: sun, rebirth after fire, creative force

### Role coverage within Fire
| Role    | Expression                                          |
|---------|-----------------------------------------------------|
| Attack  | Burn stack application + detonation                 |
| Defense | Reactive fire on being hit (counter-burn)           |
| Heal    | Solar temp STR bonus                                |
| Utility | Burn spread, stack manipulation                     |

---

## Ice — Binding x Preservation

### Primary mechanic — Frost (binding/control)
- Apply Frost stacks to enemies → SPD reduction
- At stack threshold: frozen (lose one action)
- Core feel: "slow down and lock out"

### Secondary mechanic — Preserve (protection/preservation)
- Ice as preserver, not only restraint
- Expressed as: AMR reinforcement, damage mitigation for allies, crystalline shielding
- Thematic basis: cryogenic preservation, clarity, stillness as strength

### Role coverage within Ice
| Role    | Expression                                                    |
|---------|---------------------------------------------------------------|
| Attack  | Frost damage + slow application                               |
| Defense | Preserve AMR boost, damage mitigation                         |
| Heal    | Damage prevention by freezing enemies before they act         |
| Utility | SPD manipulation, Frost stack control                         |

---

## Earth — Resistance x Nurturing

### Primary mechanic — Guard (resistance/counter)
- Accumulate Guard stacks to raise AMR
- On being hit: return counter-damage proportional to Guard stacks
- Core feel: "tank hits then punish"

### Secondary mechanic — Growth (nurturing/sustain)
- Earth as life-giver, not only immovable wall
- Expressed as: gradual HP recovery (temp Strength over time), propagating buffs to allies
- Thematic basis: fertile ground, roots, the earth sustaining all life

### Role coverage within Earth
| Role    | Expression                                            |
|---------|-------------------------------------------------------|
| Attack  | Guard stack consumption → heavy strike                |
| Defense | Guard stack accumulation                              |
| Heal    | Growth propagation, gradual temp STR recovery         |
| Utility | Ally buff spread, positioning effects                 |

---

## Cross-Element Design Note

All three elements use the same design language: **stack accumulation → threshold effect**. This shared grammar means:
- Players learn the system once and apply it across elements
- Multi-element builds can manage multiple stacks simultaneously
- Stack thresholds create natural decision points each turn
